# Go APM Server UI Fix Tasks

## 진행 현황

- 완료: 11 / 23
- 현재 기준 완료 범위: Phase 1, Phase 2 (`UIFIX-001` ~ `UIFIX-011`)

## 작업 원칙

- 깨지는 동선 제거를 가장 먼저 처리한다.
- 상태 처리와 반응형 구조를 공통화한 뒤 화면별 세부 개선으로 들어간다.
- 각 단계 종료 시 로컬 UI 확인 기준을 함께 검증한다.
